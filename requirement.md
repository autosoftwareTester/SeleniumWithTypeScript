Initial SetUp
=============
npm init -y in your project folder.
Install Selenium: Run npm install selenium-webdriver 
Install Selenium: Run npm install --save-dev @types/selenium-webdriver
Install Development Dependencies: Run npm install --save-dev typescript ts-node @types/node.
Initialize TypeScript: Use npx tsc --init to create a tsconfig.json file.

Run a test
==============
npx ts-node .\test\upload-file.ts
npx ts-node .\test\com.company.qa\exercises\webTable.ts
